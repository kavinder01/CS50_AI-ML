# CS50_AI-ML
--CS50’s Introduction to Artificial Intelligence with Python
By Harvard University

This repository contains my complete implementations of projects from CS50 AI, where I explored fundamental AI concepts including search, knowledge representation, uncertainty, optimization, machine learning, and neural networks.

Each project demonstrates practical implementation of core AI algorithms using Python.

📌 1️⃣ Search
🔹 Tic-Tac-Toe

Implemented the Minimax algorithm to create an optimal AI agent that never loses.
Explored:

Game trees

Recursive decision making

Adversarial search

🔹 Knights

Built a knowledge-based system using propositional logic to solve Knights and Knaves logical puzzles.
Focused on:

Logical inference

Model checking

Knowledge representation

🔹 Minesweeper

Developed an AI agent that plays Minesweeper using logical inference and constraint satisfaction.
Implemented:

Knowledge base construction

Logical deduction

Safe move inference

📌 2️⃣ Knowledge
🔹 Heredity

Built a probabilistic model to compute gene inheritance using Bayesian probability.
Learned:

Joint probability

Conditional probability

Genetic trait modeling

🔹 PageRank

Implemented Google’s PageRank algorithm using:

Sampling method

Iterative method

Explored:

Markov chains

Probability distribution convergence

📌 3️⃣ Uncertainty
🔹 Crossword

Solved crossword puzzles using Constraint Satisfaction Problems (CSP).
Implemented:

Node consistency

Arc consistency (AC-3)

Backtracking search

Heuristics (MRV, Degree heuristic, LCV)

📌 4️⃣ Optimization
🔹 Nim

Built a reinforcement learning agent using Q-learning that improves by self-play.
Concepts applied:

Epsilon-greedy strategy

Q-value updates

Reinforcement learning

📌 5️⃣ Learning
🔹 Shopping

Implemented a machine learning classification model to predict online shopping revenue.
Used:

k-Nearest Neighbors (kNN)

Sensitivity & specificity evaluation

📌 6️⃣ Neural Networks
🔹 Traffic

Built a deep learning model using Convolutional Neural Networks (CNNs) to classify traffic signs.
Applied:

TensorFlow / Keras

Image preprocessing

Model training & evaluation
